# Base Ecosystem Research

Personal research, notes, and documentation on the Base blockchain ecosystem — covering DeFi, AI agents, social, NFTs, and airdrop opportunities.

## About Base

Base is an Ethereum Layer 2 network incubated by Coinbase, launched on mainnet in August 2023. It has rapidly grown to become one of the largest L2s by TVL, with $4.6B+ in total value locked and 1M+ active addresses.

**Key stats:**
- Chain ID: 8453
- Built on: OP Stack (Optimism)
- Native bridge: bridge.base.org
- Block explorer: basescan.org
- RPC: https://mainnet.base.org

## Why Base?

Base is the blockchain for global finance. Fast, cheap, always-on, and backed by Coinbase's distribution and trust. No native token (yet) — which makes on-chain activity the primary signal for future rewards.
