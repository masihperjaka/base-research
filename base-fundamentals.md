# Base Chain - Fundamentals

## What is Base?

Base is an Ethereum Layer 2 blockchain incubated by Coinbase, built on the OP Stack. It launched on mainnet in August 2023 and has grown into one of the most active L2s by TVL, transaction count, and developer activity.

**Tagline:** "The blockchain for global finance."

## Technical Foundation

### OP Stack
- Base is built on the OP Stack, the same framework powering Optimism
- Part of the Superchain ecosystem — interoperable with OP chains
- EVM-equivalent: all Ethereum tooling works natively

### Key Specs
- Chain ID: 8453
- Block time: ~2 seconds
- Gas fees: fractions of a cent per transaction
- Finality: L1 finality through Ethereum
- RPC: https://mainnet.base.org

### Base Azul (April 2026)
- Latest major upgrade to Base's tech stack
- Makes Base more secure, more performant, and easier to build on
- Part of the "Unified Base Chain Stack" evolution toward further decentralization

## Why Base Stands Out

1. **Coinbase Distribution** — Direct access to 100M+ Coinbase users
2. **No Native Token (yet)** — On-chain activity is the main signal for future eligibility
3. **Institutional Trust** — Backed by a publicly traded company (Coinbase/COIN)
4. **Low Fees** — Sub-cent transactions make micro-interactions viable
5. **Fast Finality** — Sub-second confirmation for most use cases

## Key Infrastructure

### Base Account
- Unified auth layer (passkeys, smart wallets)
- Basenames (human-readable .base.eth addresses)
- Built-in payments infrastructure

### Base App (formerly Coinbase Wallet)
- "Super app" integrating wallet, trading, DeFi, social
- Integrated Zora and Farcaster in July 2025
- Base Verify for identity verification

### Base Bridge
- Official bridge: bridge.base.org
- Supports ETH and ERC-20 tokens
- December 2025: Solana bridge launched via Chainlink CCIP

## Ecosystem Size (2025)
