# Farcaster — Decentralized Social on Base

## Overview

Farcaster is a decentralized social protocol built on Base, often described as the "crypto-native Twitter." It enables censorship-resistant social networking where users own their social graph and data.

**Primary client:** Warpcast (warpcast.com)
**Website:** farcaster.xyz
**Chain:** Base

## How It Works

Farcaster is a **sufficiently decentralized** social network:
- User accounts are stored on-chain (Base)
- Content (casts) are stored off-chain on "Hubs" (decentralized nodes)
- Any developer can build a client on top of the protocol
- Users own their social graph — portable across clients

### Key Components
- **Farcaster ID (FID)** — on-chain identifier
- **Casts** — posts (equivalent of tweets)
- **Channels** — topic-based communities
- **Frames** — interactive mini-apps embedded in casts
- **Mini Apps** — full apps running inside Farcaster clients

## Key Integrations

### Clanker Acquisition (October 2025)
Farcaster acquired Clanker, the AI-powered token deployment bot. Users can tag @clanker on Warpcast to instantly deploy an ERC-20 token on Base. This turned Farcaster into a social token launchpad.

### Base App Integration (July 2025)
The Base App (formerly Coinbase Wallet) integrated Farcaster alongside Zora, placing decentralized social at the center of Base's consumer super app.

## Why Farcaster Matters for Base

1. **Social graph is on-chain** → activity signals identity and reputation
2. **Frames enable onchain actions** inside social posts → DeFi, NFT mints, etc.
3. **Creator economy** — Clanker + Zora integration creates social tokens
4. **Base-native community** → highest concentration of crypto-native builders

## How to Get Started

1. Create account at warpcast.com
2. Register a Farcaster ID (small ETH fee on Optimism/Base)
3. Start casting, follow channels, engage with mini apps
4. Use Frames to interact with DeFi and NFT protocols
